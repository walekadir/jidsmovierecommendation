What to Watch Tonight — AI Movie Recommender

An AI-powered, lightning-fast movie and TV show recommendation engine. Built as part of [Jid's Personal Dashboard], this tool cures "decision fatigue" by generating highly personalized watchlists based on your exact mood and preferences.

The Problem & Solution
The Problem: Spending 45 minutes scrolling through streaming platforms just to rewatch a show you've already seen five times. 

The Solution: A smart interface that takes your mood, preferred genres (including specialized niches like Nollywood and Anime), and custom context to generate a curated list of 5 movies in seconds. 

Features

Mood-Based Logic: Recommends content based on how you feel (e.g., "Mind-bending", "Just chill", "Intense").

Global & Local Cinema: Native support and specific prompts for Hollywood, Nollywood, K-Drama, and Anime.

Hidden Gems:*An option to explicitly hide mainstream/blockbuster picks to help you discover underrated cinema.

Blazing Fast: Powered by the Groq API (Llama 3.3 70B), returning complex JSON responses almost instantly.

Beautiful UI: Built from scratch with a custom "film grain" aesthetic, projector beam lighting, and smooth CSS animations.

Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript
AI / Backend: [Groq API](https://groq.com/) using the `llama-3.3-70b-versatile` model.
Deployment: Vercel / GitHub Pages
